# 🎯 布林带自动双向策略

## ✨ 核心特性

**一个配置，自动双向交易**

```json
{
  "strategy_mode": "bidirectional",
  "long_entry_conditions": [
    {"type": "BOLL", "op": "touch_lower", "period": 20}
  ],
  "short_entry_conditions": [
    {"type": "BOLL", "op": "touch_upper", "period": 20}
  ]
}
```

系统自动识别这是布林带双向策略，启用**智能模式**：

- 📉 **触及下轨** → 自动做多
- 📈 **触及上轨** → 自动做空
- 🎯 **回到中轨** → 自动平仓
- 🛡️ **触发止损/止盈** → 强制平仓

## 📊 工作原理

### 1. 布林带计算

```
周期 = 20 根 K 线
中轨 = 20 周期移动平均线
上轨 = 中轨 + 2 × 标准差
下轨 = 中轨 - 2 × 标准差
```

### 2. 交易信号

```
价格走势图:

68000 ┤                 ╱╲         ← 上轨（触发做空）
      │              ╱    ╲ 🔴
67000 ┤         ╱╲ ╱      ╲╱╲      ← 中轨（平仓）
      │      ╱    ╲         ╲  🎯
66000 ┤   ╱        ╲          ╲╱   ← 下轨（触发做多）
      │🟢
      └────────────────────────────
       做多入场        做空入场
```

### 3. 仓位管理

- 每个方向使用 `risk_per_trade / 2` 的风险
- 做多做空可以**同时持仓**
- 互不影响，独立管理

## 💡 优势特点

### ✅ 相比手动配置

**旧方式**：需要分别配置做多、做空、平仓条件
```json
{
  "long_entry_conditions": [...],    // 做多入场
  "short_entry_conditions": [...],   // 做空入场
  "exit_conditions": [...],          // 平仓条件
}
```

**新方式**：自动识别，一键配置
```json
{
  "long_entry_conditions": [
    {"type": "BOLL", "op": "touch_lower", "period": 20}
  ],
  "short_entry_conditions": [
    {"type": "BOLL", "op": "touch_upper", "period": 20}
  ]
}
// 系统自动处理：触及中轨平仓
```

### ✅ 智能特性

1. **自动识别布林带双向模式**
   - 当两个条件都只有一个 BOLL 条件时
   - 且分别是 touch_lower 和 touch_upper
   - 系统自动启用智能模式

2. **自动平仓**
   - 价格回到中轨 ± 0.2% 时自动平仓
   - 无需手动配置退出条件

3. **灵活触发**
   - 允许 0.2% 的触及误差
   - 避免因小幅度回调错过入场机会

## 💬 常见问题

### Q1: 为什么叫"自动"双向？
**A**: 系统识别到是布林带上下轨配置时，会自动：
- 触及下轨 → 做多
- 触及上轨 → 做空
- 回到中轨 → 平仓

无需手动配置退出条件。

### Q2: 能同时持有多空仓位吗？
**A**: 可以！做多和做空独立管理，互不影响。

### Q3: 如何判断是否触及布林带？
**A**: 允许 0.2% 误差：
- 触及下轨：`价格 <= 下轨 × 1.002`
- 触及上轨：`价格 >= 上轨 × 0.998`

### Q4: 必须回到中轨才平仓吗？
**A**: 不是，以下情况也会平仓：
- 触发止损
- 触发止盈
- 回到中轨 ± 0.2%

### Q5: 能添加其他条件过滤吗？
**A**: 可以！但会退出自动模式，需要手动配置所有条件。
