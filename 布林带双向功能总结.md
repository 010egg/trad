# 🎉 布林带自动双向交易 - 功能总结

## ✨ 核心特性

### **智能识别，自动双向**

当你配置：
```json
{
  "strategy_mode": "bidirectional",
  "long_entry_conditions": [
    {"type": "BOLL", "op": "touch_lower", "period": 20}
  ],
  "short_entry_conditions": [
    {"type": "BOLL", "op": "touch_upper", "period": 20}
  ]
}
```

系统**自动识别**这是布林带双向策略，并启用智能模式：

✅ 📉 触及下轨 → **自动做多**
✅ 📈 触及上轨 → **自动做空**
✅ 🎯 回到中轨 → **自动平仓**
✅ 🛡️ 止损止盈 → **自动执行**

---

## 🔥 关键改进

### 1. 智能识别算法

```python
def _is_boll_bidirectional_mode(long_entry, short_entry):
    """

    - 分别是 touch_lower 和 touch_upper

    """
```

### 2. 自动平仓逻辑

```python
def _check_boll_signal(indicator_data, current_price, i):
    """
    实时检测布林带信号：
    - touch_lower: 价格 <= 下轨 × 1.002
    - touch_upper: 价格 >= 上轨 × 0.998
    - middle: 价格在中轨 ± 0.2% 范围
    """
```

### 3. 容错机制

- 允许 **0.2% 的触及误差**
- 避免因小幅波动错过入场
- 中轨平仓也有 0.2% 容差

## 💡 使用技巧

### 1. 市场判断

使用前先判断市场状态：
```python

# 看 ADX（趋势强度）
# 看价格是否在区间内震荡

    使用布林带双向策略 ✅
else:  # 有明显趋势
    改用趋势跟踪策略 ⚠️
```

### 2. 参数优化

回测不同周期：
```python

选择最优参数

    {"type": "RSI", "op": "lt", "value": 35}
  ]
}
```
