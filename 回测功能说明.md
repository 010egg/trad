# 🎯 TradeGuard 回测功能说明

## ✨ 新增功能

### 1. 双向交易支持

现在支持三种策略模式：

- **long_only**: 仅做多（默认）
- **short_only**: 仅做空
- **bidirectional**: 双向交易（震荡策略）

### 2. 震荡策略

专为震荡市场设计的双向交易策略：

- 📉 **逢低做多**: 价格触及布林带下轨时开多单
- 📈 **逢高做空**: 价格触及布林带上轨时开空单
- 🎯 **止盈**: 价格回归布林带中轨时平仓
- 🛡️ **止损**: 设定的百分比止损

## 🎨 策略预设

系统内置了三个策略预设：

### 1. 布林带震荡策略 (oscillation)
```json
{
  "strategy_mode": "bidirectional",
  "entry_conditions": [{"type": "BOLL", "op": "touch_lower", "period": 20}],
  "exit_conditions": [{"type": "BOLL", "op": "touch_upper", "period": 20}],
  "stop_loss_pct": 3.0,
  "take_profit_pct": 5.0,
  "recommended_interval": "15m",
  "tips": "适合横盘震荡行情，趋势市场慎用"
}
```

### 2. MA趋势跟踪 (trend_following)
```json
{
  "strategy_mode": "long_only",
  "entry_conditions": [{"type": "MA", "fast": 20, "slow": 60, "op": "cross_above"}],
  "exit_conditions": [{"type": "MA", "fast": 20, "slow": 60, "op": "cross_below"}],
  "stop_loss_pct": 2.0,
  "take_profit_pct": 8.0,
  "recommended_interval": "1h",
  "tips": "适合趋势明显的行情"
}
```

### 3. RSI超买超卖反转 (rsi_reversal)
```json
{
  "strategy_mode": "bidirectional",
  "entry_conditions": [{"type": "RSI", "period": 14, "op": "lt", "value": 30}],
  "exit_conditions": [{"type": "RSI", "period": 14, "op": "gt", "value": 70}],
  "stop_loss_pct": 2.5,
  "take_profit_pct": 5.0,
  "recommended_interval": "15m",
  "tips": "适合震荡市，需注意假突破"
}
```

## 📊 回测指标说明

| 指标 | 说明 |
|------|------|
| 总收益率 | (期末资金 - 期初资金) / 期初资金 × 100% |
| 胜率 | 盈利交易次数 / 总交易次数 × 100% |
| 盈亏比 | 总盈利 / 总亏损 |
| 最大回撤 | 从峰值到谷值的最大跌幅百分比 |
| 夏普比率 | (平均收益 - 无风险收益) / 收益标准差 |
| 平均持仓时长 | 所有交易的平均持仓小时数 |

## 🔧 高级配置

### 自定义入场条件

支持的条件类型：
- **MA/EMA**: 移动平均线
  - `op`: cross_above（金叉）, cross_below（死叉）
  - `fast`, `slow`: 快线和慢线周期

- **BOLL**: 布林带
  - `op`: touch_lower（触及下轨）, touch_upper（触及上轨）
  - `period`: 周期，默认20

- **RSI**: 相对强弱指标
  - `op`: gt（大于）, lt（小于）
  - `value`: 阈值（30/70）
  - `period`: 周期，默认14

- **KDJ**: 随机指标
  - `op`: cross_above, cross_below
  - `line`, `target_line`: K, D, J线

- **MACD**: 指数平滑异同移动平均线
  - `op`: cross_above（DIF上穿DEA）, cross_below

### 示例：组合条件

```json
{
  "entry_conditions": [
    {"type": "BOLL", "op": "touch_lower", "period": 20},
    {"type": "RSI", "op": "lt", "value": 30, "period": 14}
  ],
  "exit_conditions": [
    {"type": "BOLL", "op": "touch_upper", "period": 20}
  ]
}
```

## ⚠️ 注意事项

1. **震荡策略适用场景**
   - ✅ 横盘震荡市场
   - ✅ 有明显支撑阻力位
   - ❌ 强趋势单边市场

2. **风险管理**
   - 建议 `risk_per_trade` 不超过 2-3%
   - 双向交易时每个方向使用一半仓位
   - 设置合理的止损止盈比例

3. **回测局限性**
   - 未包含滑点和手续费
   - 历史表现不代表未来收益
   - 建议小仓位实盘验证
